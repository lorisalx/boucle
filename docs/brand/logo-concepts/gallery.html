Boucle

Boucle is French for loop — the recurring agent cycle the whole product is built around. The six directions below all draw from that one idea, in a single warm stroke plus the ember gradient already used in the deck: some read it literally (a tied loop, an infinity cycle, a ticket on a ring), others more figuratively (a curl, a monogram, a wordmark).

Knotted Loop

An open ring tied off with a small knot — the most literal boucle: a loop of string, tied. Reads well as a favicon down to 16px.

Infinity Loop

The cycle never stops — capture → route → enrich, continuously. Most abstract mark of the set; scales cleanly to a large hero size.

B Monogram

A lowercase b whose bowl is the same knotted-loop device as the first mark — ties the family together while giving a letterform anchor.

Ticket Loop

A claim-check on a ring — the chief-of-staff ticket queue, literally looped. The most "product" mark: names what Boucle manages, not just what it does.

French Curl

The other meaning of boucle — a curl. A single calligraphic stroke that spirals in on itself. Warmest, least "software" mark of the set.

Wordmark Lockup

The infinity icon paired with a lowercase, geometric-sans boucle — for headers, README badges, and the browser tab title.
